### 二分查找

```c++
在 范围a[left]到a[right]中寻找key
	
	while (left <= right)
	{
		mid = (left + right) / 2;
		if (a[mid]<key)
		{
			left = mid + 1;
          //丢弃左边
		}
		else if (a[mid]>key)
		{
			right = mid - 1;
          //丢弃右边
		}
		else
		{
			return mid;
		}
	}
```

## 查找

```c++
 auto search=award.find(9527);
    
  if(search != award.end())
 {
        std::cout << "Found " << search->first << " " << search->second << '\n';
    }
    else
 {
        std::cout << "Not found\n";
    }
}
```

## 二分搜索

Lower_bound:

参数: first last val cmp(可省略)

功能:返回从first到last过程中 第一个大于等于val的 值的索引

upper_bound:

参数: first last val cmp(可省略)

功能:返回从first到last过程中 第一个大于val的 值的索引
